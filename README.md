# PDF 보안 처리 시스템

전자책 PDF 파일에 구매자 정보를 워터마크로 삽입하고 비밀번호를 설정하여 무단 배포를 방지하는 시스템입니다.

## 주요 기능

- ✅ **워터마크 삽입**: PDF 파일에 구매자 정보 워터마크 자동 삽입
- ✅ **비밀번호 보호**: PDF 파일에 비밀번호 설정
- ✅ **메타데이터 추가**: 문서 속성에 구매자 정보 추가
- ✅ **크로스 플랫폼**: macOS 지원
- ✅ **GUI 인터페이스**: 직관적인 그래픽 사용자 인터페이스

## 다운로드

### 최신 버전 다운로드

1. [Releases](https://github.com/your-username/pdf_secure/releases) 페이지 이동
2. 최신 버전의 `PDF보안처리-macOS.zip` 다운로드
3. ZIP 파일 압축 해제
4. `PDF보안처리` 파일 실행

## 사용 방법

### 1. 실행
- 다운로드한 `PDF보안처리` 파일 더블클릭

### 2. 파일 선택
- "파일 선택" 버튼 클릭
- 원본 PDF 파일 선택

### 3. 구매자 정보 입력
- **이름**: 구매자 이름 입력
- **연락처**: 전화번호 또는 이메일 입력
- **PDF 비밀번호**: 원하는 비밀번호 입력
- 입력 시 하단에 워터마크 문구 미리보기 표시

### 4. 처리 시작
- "처리 시작" 버튼 클릭
- 진행 상황 바와 상태 메시지로 처리 과정 확인

### 5. 결과 확인
- 처리 완료 시 성공 메시지와 비밀번호 정보 표시
- 생성된 파일: 원본 파일과 같은 디렉토리에 `{원본파일명}_{구매자명}.pdf` 형식으로 저장
- 파일 열기 시 비밀번호 입력 필요
- 문서 속성에서 구매자 정보 확인 가능

## 시스템 요구사항

- macOS 10.14 이상
- Python 설치 불필요 (모든 의존성 포함)

## Gatekeeper 경고 해결

macOS에서 실행 시 "확인되지 않은 개발자" 경고가 나타날 수 있습니다.

### 해결 방법 1: 시스템 설정에서 허용
1. 시스템 설정 > 보안 및 개인 정보 보호
2. "확인" 버튼 클릭하여 허용

### 해결 방법 2: 터미널에서 실행
```bash
xattr -cr PDF보안처리
```

## 개발자 가이드

### 로컬 빌드

```bash
# 가상환경 활성화
source pdf_secure/bin/activate

# 의존성 설치
pip install -r requirements.txt

# 빌드 실행
chmod +x build.sh
./build.sh
```

### GitHub Actions 빌드

태그를 푸시하면 자동으로 빌드됩니다:

```bash
git tag v2.0.0
git push origin v2.0.0
```

자세한 내용은 [BUILD_GUIDE.md](BUILD_GUIDE.md)를 참고하세요.

- **PyInstaller**: 실행 파일 패키징
